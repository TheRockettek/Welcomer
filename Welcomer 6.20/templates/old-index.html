The go-to bot for your discord server

Welcomer is a multipurpose discord bot for your discord server with the task of providing peace of mind when opperating it. This bot brings many features such as moderation, role management, invite tracking and most notably, welcome messages, for use to simplify the opperation of managing your server and to make sure anyone can use it easily.

Features

Moderation

Moderation allowing for mod logs and timed punishments and auto moderation which is fully configurable

Dashboard

A versitile, easy to use web dashboard to allow you to easily control all features of the bot without having to memorise any commands

Welcome Images

Welcomer Images allow for customisable images ranging from many backgrounds, themes and text colours

Borderwall

Borderwall allows for easy accessibility to ensure users who join are not bots and ensure that people who join can be trusted
