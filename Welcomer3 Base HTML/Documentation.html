General

How to change the prefix

To be able to change the prefix you will need to use +setprefix <prefix> and you will need to require elevation. Please do not forget this prefix!
+setprefix >
+setprefix aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa

How to use preferdms

Prefer DMs is a feature where, if enabled, will instead send you help for usage and such in your direct messages instead of in the channels. This will send general usage help however permission errors and such will not send in your direct messages. Please note that the help command will always preferdms. To enable prefer dms just do +preferdms <True> or to disable it do +preferdms <False>. Please note if you do not specify after if you want it on or not, it will just toggle between the two.
+preferdms True
+preferdms False

How to obtain Welcomer Pro / Welcomer +

At the moment it is not possible to obtain Welcomer Pro or Welcomer +. This is because the site has not been fully done and we do not want you to have the opportunity to donate and it does not go through properly or you do not get the role.

Welcomer

How to enable/disable Welcomer

To enable welcomer, all you simply need to do is +welcomer enable, however before you do that make sure you have set the channel first and the bot has permissions to ATTACH_FILES and SEND_MESSAGES.
+welcomer enable
+welcomer disable

How to change the channel

To change the background for welcomer, all you need to do is +welcomer setbackground <channel>. You are able to select backgrounds by either entering its name, its id or mentionining it.
+welcomer set-channel #welcome
+welcomer set-channel 358943134337466371
+welcomer set-channel Welcome

How to enable/disable text/images

To toggle images just do +welcomer use-image <enable/disable>. If you do not supply either enable, disable, true, false etc. then it will just toggle between its two states.
This applies the same with +welcomer use-text <enable/disable>.
+welcomer use-text True
+welcomer use-image False

How to mention users in text

To mention users in welcome text, use the formatting tag $member_mention. Make sure you do not use this for images as images do not support the mention tag.
+welcomer set-message Welcome $member_mention!

How to change the image text

To be able to change the image text you can do +welcomer set-image-text <position> ,,,. The 3 possible positions are top, middle and bottom and you do not require to encapsulate the text in parentheses.
+welcomer set-image-text top Welcome $member
+welcomer set-image-text middle to $server!
+welcomer set-image-text bottom You are the $members user :)

How to change the background

To be able to change the background all you need to do is +welcomer set-background <background>. You are able to obtain a list of all the backgrounds with +welcomer backgrounds and the background names are case sensitive. It is possible to see how the images look beforehand by either visiting the support server in the #backgrounds channel or doing the command +welcomer preview-background <background>.

When Welcomer Pro and Welcomer + is released, it will be possible to set your custom background by using +welcomer set-custom-background then uploading an image afterwards however this is not out yet.
+welcomer set-background Tanya
+welcomer backgrounds
+welcomer preview-background Aesthetics
+welcomer set-custom-background

Where can i find the markdown

To find the markdown for welcomer text, images and leaver aspects, do +welcomer markdown.
+welcomer markdown.

General Solutions to problems


The issue of a broken image is usually caused by an invalid hex code on either the image colour or circle colour. Please ensure your hexidecimal colours are either 6 letters long or 8 letters (if using RGBA).
Good - Is either 6 or 8 letters +welcomer set-image-colour text 000000
Bad - Is not 6 or 8 letters+welcomer set-image-colour text FFF

If your welcomer is no longer welcoming people, please use +welcomer config to verify they are online and the channel exists. Usually when doing +welcomer, a list will appear at the bottom to help resolve common errors and if text or images are not enabled.

Leaver

How to enable/disable Leaving messages

To enable or disable leaving messages you need to do either +leaver enable or +leaver disable.

How to change leaver channel

To change the background for leaver, all you need to do is +leaver setbackground <channel>. You are able to select backgrounds by either entering its name, its id or mentionining it.
+leaver set-channel #goodbye
+leaver set-channel 429724829692985364
+leaver set-channel goodbye

How to change the leaver message

To change the leaver message, you can do +welcomer set-message .... Please note this can also follow the welcomer markdown and also does not require parentheses to encapsulate it.
+leaver set-message Goodbye $member, it was nice knowing you :wave:

Where can i find the markdown

To find the markdown for welcomer text, images and leaver aspects, do +welcomer markdown.
